# CalculatorProject
DNA Project for Bank Mizrahi
API for performing simple arithmetic operations (add, subtract, multiply, divide). 

## Run

Linux/OS X:

```
sh build.sh
```

Windows:

```
build.bat
```

## Run in Docker

```
cd src/IO.Swagger
docker build -t io.swagger .
docker run -p 5000:5000 io.swagger
```
